# Diagnostics Utility for Intel速 oneAPI Toolkits

Diagnostics Utility for Intel速 oneAPI Toolkits is a tool designed to diagnose the system status for using Intel速 software.

- Ubuntu 20.04 LTS

- RHEL 8.2
- RHEL 8.3
- RHEL 9.0
- SLES 15 SP2

- SLES 15 SP4
